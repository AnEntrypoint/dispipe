Install & Use

Install from npm, set three env vars, wire two functions.

$ install
npm install dispipe

Environment

DISCORD_BOT_TOKEN

Official Discord bot token (discord.js v14 compatible).

GUILD_ID

Snowflake of the target guild.

CHANNEL_ID

Snowflake of the voice channel to join.

Bot must hold CONNECT and SPEAK permissions in the target channel.

Example

import { createClient, joinDiscordVoice, subscribeToSpeaker } from 'dispipe/client'
import { initVoicePlayer, pushAudioFrame } from 'dispipe/voice'

const client = createClient()
await client.login(process.env.DISCORD_BOT_TOKEN)

const { voiceConnection } = await joinDiscordVoice(
  client,
  process.env.GUILD_ID,
  process.env.CHANNEL_ID
)

initVoicePlayer(voiceConnection)

subscribeToSpeaker(userId, (uid, f32) => {
  console.log('inbound', uid, f32.length)
})

pushAudioFrame(new Float32Array(960))

Notes

  • ▸ Standalone — no Electron, no browser-only APIs. Works in headless bots, bridges, and Electron main.
  • ▸ Frames must be exactly 960 samples (20ms @ 48kHz).
  • ▸ Float32 clipping clamps at envelope; no amplitude overflow.

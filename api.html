Module API

Two import paths. Each handles one half of the audio pipeline.

import from 'dispipe/client'

Client & Voice Join

discord.js v14 client wrapper. Voice channel join with retry/backoff and inbound speaker subscription.

createClient()

Instantiate a configured discord.js Client with Guilds + GuildVoiceStates intents.

joinDiscordVoice(client, guildId, channelId)

Join voice channel. 8-attempt retry loop with code-aware backoff (4017 → 10s, 4006 → 8s, else 4s).

subscribeToSpeaker(userId, onPcmChunk)

Subscribe to a user's inbound Opus stream. Decodes to Float32 48kHz stereo and invokes callback.

leaveVoice()

Tear down voice connection and clean up subscriptions.

import from 'dispipe/voice'

Outbound Audio Player

AudioPlayer pipeline. Float32 PCM → s16le → Opus encode → Discord voice connection.

initVoicePlayer(connection)

Wire a voice connection to the AudioPlayer. Sets up the encoder pipeline.

pushAudioFrame(f32Buffer)

Enqueue a Float32 PCM frame at 48kHz stereo. Frames must be 960 samples (20ms).

stopAudio()

Stop playback, drain encoder, release streams.
